# 🎭 Playwright UI & API Test Automation for Automation Exercise

An end-to-end test automation framework built with Playwright, following the Page Object Model (POM) design pattern. This project automates both UI and API test scenarios for the Automation Exercise website and demonstrates modern test automation best practices.

## 🚧 Status

This project is currently under development. UI and API test cases are being automated, and the documentation will be updated as the project evolves.
